# 에셋 다운로드 가이드

## 필요한 에셋

### 1. 배경 (Kenney Background Elements)
**다운로드**: https://kenney.nl/assets/background-elements

다운로드 후 다음 파일을 복사:
```
PNG/backgroundColorForest.png → public/assets/bg/far.png
PNG/backgroundColorFall.png   → public/assets/bg/mid.png
PNG/backgroundColorDesert.png → public/assets/bg/desert.png
```

### 2. 타일 (Kenney Pixel Platformer)
**다운로드**: https://kenney.nl/assets/pixel-platformer

다운로드 후 다음 파일을 복사:
```
Tiles/tile_0000.png → public/assets/tiles/ground_grass.png
Tiles/tile_0001.png → public/assets/tiles/ground_dirt.png
```
